Descripcion: menú de administrador

Se presenta una interfaz de usuario de un sistema administrativo con un diseño oscuro. A la izquierda hay un menú vertical con varias opciones y botones, como:

Registro de Usuarios (acompañado de un ícono de lápiz).

Crear Horario (con un ícono de calendario).

Editar Horario (también con un ícono de calendario).

Registro de asistencia (con un ícono de una figura humana).

Etapa de Aprendizaje (con un ícono relacionado con aprendizaje o progreso).

Solicitudes de Instructores (representado por un ícono de documentos).

Eventos (acompañado de un icono de una estrella en un cuadro)

En la parte superior, hay un encabezado con las palabras INICIO y una barra de búsqueda, junto con un ícono de campana para notificaciones. A la izquierda, se muestra el perfil del administrador con la etiqueta "ADMIN".
