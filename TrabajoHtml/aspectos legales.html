4. Aspectos legales (normas o leyes)


El sistema debe contar con todas las leyes y regulaciones aplicables, incluyendo las relacionadas con protección de datos, propiedad intelectual y seguridad informática

  • Ley de protección de datos personales
  • Ley de propiedad intelectual
  • Normas laborales relacionadas con el uso de tecnología
  • Contratos de licencia de software5

5. Restricciones del software

  • Muchas personas ingresando a la vez, podrían hacer caer el servidor.
  • Tiempo de respuesta
  • Encriptación para ciertos tipos de datos
  • Sistemas operativos.
