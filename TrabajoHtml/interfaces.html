INTERFACES DEL USUARIO

1. Interfaz para Aprendices

Objetivo: Facilitar la consulta de horarios, registro de asistencias y acceso a información sobre la etapa productiva.

Elementos clave:

  • Mis horarios: Acceso directo al calendario personal.
  • Eventos: Visualización de actividades académicas y cambios relevantes.
  • Asistencias: Registro y justificación de inasistencias.
  • Etapa productiva: Para aprendices en esta fase, acceso a bitácoras y reportes.
  • Barra de búsqueda: Para localizar eventos o clases específicas.

2. Interfaz para Instructores

Objetivo: Permitir la gestión de horarios, registro de asistencias y monitoreo de estudiantes.

Elementos clave:

  • Calendario académico: Visualización detallada del cronograma institucional.
  • Eventos del mes: Listado de actividades y cambios programados.
  • Solicitar cambio de horario: Formulario dinámico para modificaciones.
  • Registro de asistencias: Herramienta para gestionar la asistencia de los aprendices.
  • Seguimiento de etapa productiva: Evaluación y revisión de bitácoras de aprendices.

3. Interfaz para Administradores

Objetivo: Facilitar la creación, modificación y supervisión de horarios, usuarios y solicitudes.

Elementos clave:

  • Gestión de horarios: Creación, edición y asignación de horarios.
  • Solicitudes de instructores: Módulo para administrar peticiones de cambios de horarios.
  • Registro de usuarios: Alta y baja de aprendices e instructores con opciones de importación masiva.
  • Reportes y estadísticas: Acceso a métricas sobre uso de recursos y desempeño académico.
